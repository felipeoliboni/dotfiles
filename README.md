# Dotfiles

My personal dotfiles for Linux (Hyprland). Managed with [yadm](https://yadm.io) for easy symlinking.


## Contents

- Shell: Zsh with [Oh My Zsh](https://ohmyz.sh/) + custom plugins
- Editor: NeoVim
- Terminal: kitty
- Hypr

## Installation

### Prerequisites
- yadm (`pacman -S yadm`)
- git
